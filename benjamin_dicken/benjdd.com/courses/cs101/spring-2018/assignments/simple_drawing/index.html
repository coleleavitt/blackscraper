CSc 101: Simple Drawing

This assignment should be written in the processing programming language. You should only use processing features that we have discussed up to this point. If you have questions about whether or not a particular feature can/should be used, ask the instructors! You may use any processing version 3.0 or newer.

In this assignment, you’ll be writing a few small and simple processing programs, that display simple shapes and drawings.

Four Edges

Write a processing program that displays a canvas that looks just like the following.

A few guidelines:

  • The window should be 400 pixels wide and 400 pixels tall
  • Each shapes should show up in the positions as show
    • The shapes should not be touching the edge of the canvas (they should have about a 20-pixel buffer)
    • The shapes may not touch eachother (except for the triangle tips)

Name this processing program four_edges.pde.

Target

Write a processing program that displays a canvas that looks just like the following.

A few guidelines:

  • The window should be 400 pixels wide and 400 pixels tall
  • The edge of the target should not touch the edge of the screen, but should be close!
  • You should use three ellipses and one rectangle

Name this processing program target.pde.

Funky

Write a processing program that displays a canvas that looks just like the following.

A few guidelines:

  • The window should be 300 pixels wide and 300 pixels tall
  • You should use one ellipse, one rect, and one triangle.
  • The shape should be centered on the screen, and all shapes should be the same size / position as in the image.

Name this processing program funky.pde.

Tic Tac Toe Pic

Write a processing program that displays a canvas that looks just like the following.

A few guidelines:

  • The window should be 600 pixels wide and 600 pixels tall
  • The order that you draw the shapes in is important!
  • The bars that make the board should be rectangles, not lines!
  • The vertical bars should be on-top-of the horizontal ones
  • There should be two Xs (drawn with lines) and two Os (drawn with an ellipse) on the board, in the same locations as what is shown in the picture.

Name this processing program tic_tac_toe_pic.pde.

CSc 101: Basic Shapes

This assignment should be written in the processing programming language. You should only use processing features that we have discussed up to this point. If you have questions about whether or not a particular feature can/should be used, ask the instructors! You may use any processing version 3.0 or newer.

In this assignment, you’ll be writing a few small and simple processing programs, that display basic images and shapes.

Four Corners

Write a processing program that displays a canvas that looks like the following.

A few guidelines:

  • The window should be 400 pixels wide and 400 pixels tall
  • Each shapes should show up in the positions as show
    • The shapes should not be touching the edge of the canvas (they should have about a 20-pixel buffer)
    • The shapes may not touch eachother

Name this processing program four_corners.pde.

Points

Write a processing program that displays a canvas that looks like the following.

A few guidelines:

  • The window should be 300 pixels wide and 300 pixels tall
  • The tips of the arrows (triangles) should touch the edge of the screen
  • The edges of the shapes should meet with eachother as pictured
  • You should use two triangles, one ellipse, and one rect

Name this processing program points.pde.

Face

Write a processing program that displays a canvas that looks like the following.

A few guidelines:

  • The window should be 400 pixels wide and 300 pixels tall
  • You should use three ellipses and one triangle
  • The face should be centered on the screen
  • The eyes and mouth should be displayed in roughly the same position as the image above.

Name this processing program face.pde.

Tree

Write a processing program that displays a canvas that looks like the following.

A few guidelines:

  • The window should be 300 pixels wide and 600 pixels tall
  • The order that you draw the shapes in is important!
  • Each “section” of the tree should get progressively wider as it gets closer to the base
  • The top three sections should all have 1 circular ornament (20x20 pixels)
  • The bottom section should have 2 circular ornaments (20x20 pixels)

Name this processing program tree.pde.

Problem 3

Write a program named word-lengths.py. This program will do the following:

  • First, ask the user for the name of an input file to do regex processing on.
  • Read in the file and save the contents into a string.
  • As the user for a number
  • Using regular expressions, find all words in the input file that are of exactly the specified length
  • Report the number of matching words to the user

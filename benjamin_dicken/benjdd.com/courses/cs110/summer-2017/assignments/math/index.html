Math (X points)

In this assignment, you will write multiple small programs that compute various mathematical formulas.

Volume

Write a python program and name it volume.py. This program will be used to compute the volume of a cube. Recall that the volume of a cube is the width x height x length.

At the beginning of your program, you should ask the user of your program (you!) for three numeric values.

Specifically, you should ask for the width, height, and length of a cube. When run, the part of the program that requests these inputs should look like this:

Enter width: X
Enter height: Y
Enter length: Z

The user will be able to put any positive integer number for X, Y, and Z.

Next, you must write a function named volume that computes the volume of a cube given the dimensions stores in these variables. It should then print out:

The volume of a cube with width X, height Y, and length Z is A

(But you should replace X, Y, Z, and A with the width, height, length, and computed result, respectively).

Pythagorean

Write a python program and name it pythagorean.py. This program will be used to compute the length of the hypotenuse of a right triangle, given the length of the other two sides. Recall the Pythagorean theorem, which states that:

a^2 + b^2 = c^2

For right triangles.

At the beginning of your program, you should ask the user of your program (you!) for two numeric values.

Specifically, you should ask for the lengths of the two sides of the triangle that make up the right angle. When run, the part of the program that requests these inputs should look like this:

Enter first side: X
Enter second side: Y

Again, the user will be able to put any positive integer number for X and Y.

After declaring these variables, you must write a function named print_hyp_length that computes the length of the hypotenuse, given the lengths of these two other sides. It should then print out:

The hypotenuse of a right triangle is length X when the other two sides are of length Y and Z

(But you should replace X, Y, and Z with the actual lengths).

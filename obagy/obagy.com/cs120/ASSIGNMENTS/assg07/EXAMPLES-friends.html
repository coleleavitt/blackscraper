CSc 120: Friends Examples

Input file information

The examples below are based on input files of various different sizes:

  1. in02.txt : 2 lines
  2. in04.txt : 4 lines
  3. in05.txt : 5 lines
  4. in10.txt : 10 lines
  5. in25.txt : 25 lines

CSc 120: Classes: Color

Expected Behavior

The class Color represents a color using the RGB color model. (See the following reference for information on the RGB color model: https://en.wikipedia.org/wiki/RGB_color_model .) The class Color stores the red, green, and blue components of a color as integers. Each integer is in the range of range of 0 to 255 (inclusive).

Write a Python class Color that meets the following requirements:

Attributes:

  • _r: an integer in the range of 0 to 255 (inclusive)
  • _g: an integer in the range of 0 to 255 (inclusive)
  • _b: an integer in the range of 0 to 255 (inclusive)

Methods:

  • __init__(self, r, g, b): initializes the _r, _g, and _b attributes of the object with the integers r, g, and b supplied as the method's arguments. If any of the values is less than 0, the attribute should be set to 0, and if any of the values is greater than 255, the attribute should be set to 255.

    Requirement: You must use a helper method to contain the common code that checks that each argument is in range and returns the correct value if the argument is out-of-range.

  • get_rgb(self): returns the RGB integer attributes of the object as a tuple.

  • remove_red(self): sets the attribute representing the red component of the object to zero. Leaves the green and blue attributes unchanged.

  • __str__(self): returns a string of the RGB integer attributes of the object in a specific format. For an object created by the expression Color(20,150, 200), the string returned would be
    "Color(20,150,200)"
  • A method that takes a Color object other as argument and returns True if other is the same color, and False otherwise. Two colors are equal if their corresponding RGB attribute values are equal.

    Given two Color objects c1 and c2, the following expression should determine if the colors are equal:

    c1 == c2

Examples

  1. color1 = Color(50,100,140)
    color1.get_rgb()

    Result: (50, 100, 140)

  2. color1 = Color(50,100,140)
    str(color1)

    Result: "Color(50,100,140)"

  3. color1 = Color(135,50,140)
    color1.remove_red()
    str(color1)

    Result: "Color(0,50,140)"

  4. color1 = Color(50,100,140)
    color2 = Color(135,50,140)
    color1 == color2

    Result: False

  5. color1 = Color(65,100,180)
    color2 = Color(65,100,180)
    color1 == color2

    Result: True

  6. color1 = Color(255,255,255)
    color1.remove_red()
    str(color1)

    Result: "Color(0,255,255)"

  7. color1 = Color(255,255,255)
    color1.remove_red()
    color1.get_rgb()

    Result: (0,255,255)

  8. color1 = Color(50,100,140)
    color1.get_rgb()[2]

    Result: 140
